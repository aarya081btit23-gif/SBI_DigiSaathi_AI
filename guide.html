Digital Onboarding Journey

Opening your Recurring Deposit

DigiSaathi converts the recommendation into simple, confidence-building instructions for mobile banking adoption.

Guided Journey

Open the official YONO SBI app

Use only the official app from authorized app stores. Login using MPIN/User ID without sharing credentials with anyone.

Go to the Deposits section

From the main dashboard, tap the Deposits tile to view saving options such as FD and RD.

Select Open Recurring Deposit

Choose Recurring Deposit for monthly savings. Enter monthly amount, tenure and linked savings account.

Review before confirming

Check amount, tenure, nominee and maturity details carefully. Do not continue if the page looks unofficial.

Set monthly reminder

DigiSaathi can create a reminder so the customer develops a consistent saving habit.

Need safety help?

Voice Guide

Explain it simply

For first-time or elderly users, the assistant can read each step aloud in English or Hindi and explain banking terms in simple words.


Hindi support

“हर महीने ₹1000 बचाने के लिए Recurring Deposit एक सुरक्षित विकल्प है.”

Contextual nudges

If the user pauses, the assistant can ask: “Do you want me to explain this step?”

Branch assist fallback

If the customer feels unsure, DigiSaathi suggests visiting the nearest branch or calling official SBI support.
