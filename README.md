# SBI DigiSaathi AI Prototype

Static connected prototype for SBI Hackathon Problem Statement 2: Digital Adoption.

## Pages
- `index.html` — Landing page and solution overview
- `chat.html` — AI conversational onboarding demo
- `recommendation.html` — Personalized SBI Recurring Deposit recommendation
- `guide.html` — Step-by-step digital adoption guidance
- `safety.html` — Safety, compliance and support guardrails

## How to run
Open `index.html` in any browser.

## How to deploy on GitHub Pages
1. Upload all files in this folder to a GitHub repository.
2. Go to Repository Settings → Pages.
3. Select branch `main` and root folder `/`.
4. Open the generated GitHub Pages link.

## Notes
This is a hackathon prototype. It does not connect to real SBI systems and does not process real banking transactions.
